五福 ETF V8 迭代与本地回测分析

本轮目标

本轮把同花顺弱市状态从“聚宽日志回放”推进到“本地指数自动计算优先”，同时处理执行层 100 份取整、碎股、资金不足和成本预留规则，并输出新版同花顺、聚宽和本地回测脚本。

中证 A500 数据源结论

公开口径确认：中证 A500 指数代码为 000510；同花顺行情中心的中证 A500 页面 URL 使用 1B0510。

V8 同花顺脚本采用候选顺序：000510.SH -> 000510.XSHG -> 000510.CSI -> 000510 -> 1B0510 -> 1B0510.SH。运行时会输出 WUFU_INDEX_RESOLVE，确认实际可用源；若有效指数不足 3 个，会输出 WUFU_WEAK_FALLBACK 并回退到 V7 弱市区间。

平台脚本改动

同花顺 V8

  • 默认启用自动弱市计算：USE_EXTERNAL_WEAK_STATES = False。
  • 保留聚宽日志弱市兜底：FALLBACK_EXTERNAL_WEAK_STATES = True。
  • 中证 A500 增加 1B0510 / 1B0510.SH。
  • 初始化日志升级为 WUFU_THS_FAST_V8。
  • WUFU_ORDER_PLAN 增加 estimated_cost / residual_cash / round_lot。

聚宽 V8

  • 初始化日志升级为 WUFU_JQ_FIXED_POOL_V8。
  • 弱市指数继续使用 000300.XSHG / 399101.XSHE / 399006.XSHE / 000510.XSHG。
  • 下单计划字段与同花顺一致，买入预算加入成本预留并按 100 份向下取整。

本地执行版回测

本地脚本：reports/run_wufu_v8_local_execution_backtest.py。规则为 T 日信号、T+1 日执行，佣金 0.01%、滑点 0.01%、最低佣金 5 元，保留 0.2% 现金，按 100 份向下取整。

与 V7 平台验收目标的关系

V7 已经达到弱市匹配率 100%、目标匹配率 98.48%。V8 的目标是在不破坏 95%+ 匹配率的前提下，完成同花顺 A500 自动源探测和执行层规则统一。下一轮平台验收要重点看 WUFU_INDEX_RESOLVE、WUFU_WEAK_FALLBACK 和新 WUFU_ORDER_PLAN 字段。
