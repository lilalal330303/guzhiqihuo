# 模拟盘钛银冰蓝亮色主题设计

日期：2026-07-14

## 目标

为现有“量化作战中枢”增加与深色主题对应的高质感亮色表现。亮色版应适合白天长时间查看，在强环境光下保持数据清晰，同时延续深色版的 HUD、终端与金融科技气质。

本轮只调整亮色主题的 CSS 和相关静态测试，不修改策略代码、策略参数、行情快照、账户计算、图表数据或交易逻辑。

## 视觉方向

采用“钛银冰蓝”方案：

- 页面底色使用雾银灰、冷白和极淡冰蓝环境光，避免大面积纯白。
- 顶栏、侧栏和底部移动导航使用半透明钛银玻璃效果。
- 面板采用珍珠白到冷银灰的渐变，并使用细金属描边、顶部高光和克制阴影建立层次。
- 主强调色为冰蓝，辅助色为青色；正负收益颜色继续保持清晰可辨。
- 坐标轴、正文和表格文字使用深海军蓝体系，保证白天可读性。

## 组件规则

### 页面框架

- 背景保留坐标网格，但亮色版降低对比度，不能干扰数据。
- 顶栏具有钛银玻璃渐变、下沿冰蓝高光和柔和投影。
- 侧栏与主内容区形成明确的明度分层，固定导航行为不改变。

### 策略与指标卡

- 策略卡、指标卡和内容面板使用统一的银白渐变表面。
- 边缘使用冷蓝金属描边，顶部保留细高光。
- 悬浮时轻微抬升并增强冰蓝边缘光，不使用大面积高饱和底色。
- 数字保持高对比度和等宽数字观感，不增加小数位。

### 图表

- 主曲线采用较深的冰蓝，避免白底上发虚。
- 面积填充采用低透明度蓝青渐变。
- 坐标轴和标签使用中深灰蓝。
- 十字准星、数据点和提示框与亮色主题匹配；不改变任何坐标范围或数据计算。

### 表格、活动与日志

- 表头使用冷银蓝渐变，表格行使用克制的隔行层次。
- 悬浮或选中行使用冰蓝侧光与浅蓝表面，不使用深色底板。
- 执行活动与日志使用银白卡片和冰蓝节点，保证中文内容清晰。
- 日期筛选器、下拉框、标签页和周期按钮统一为金属胶囊控件。

### 主题切换与动效

- 现有主题切换机制和用户选择记忆保持不变。
- 亮色组件切换、悬浮和高光动画控制在约 220–320ms。
- 系统启用“减少动态效果”时继续关闭非必要动画。

## 响应式设计

- 保持 360、390、430、480 像素手机宽度无横向页面溢出。
- 移动端顶栏、策略卡、指标卡、筛选器和底部导航均使用对应的钛银材质。
- 表格自身允许在容器内横向滚动，页面框架不得随表格横向移动。
- 保留安全区留白和现有触控尺寸。

## 测试与验收

- 静态测试必须验证亮色主题包含独立的钛银表面、金属边缘、图表、表格、日志和移动导航样式。
- 先新增会失败的测试，再实现样式覆盖。
- 完成后运行模拟盘静态页面测试以及盘后导出相关测试。
- 本地页面必须加载新的资源版本，避免浏览器缓存旧 CSS。
- GitHub Pages 同步后验证线上 HTML 和 CSS 已包含新版本标记。
