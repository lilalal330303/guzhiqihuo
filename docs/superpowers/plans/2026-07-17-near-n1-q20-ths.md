# near_n1_q20 同花顺 SuperMind 复刻 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 生成一份可直接粘贴到 SuperMind Python 3.8 分钟回测编辑器的 `near_n1_q20` 同花顺脚本，并完成本地兼容性验证。

**Architecture:** 单文件平台脚本采用 `init/handle_bar/before_trading/after_trading` 原生入口；选股、质量降级、信号、执行、风控和日志按函数边界拆分。平台没有财务因子时显式退化为市值临界排序，不注入本地目标数据。

**Tech Stack:** Python 3.8, pandas, numpy, SuperMind `get_price`/`get_all_securities`/`order_target_value` API。

## Global Constraints

- 交易信号只能使用 T-1 及更早数据；不得读取本地聚宽目标 CSV。
- 默认 T+1：卖出量不得超过 `closeable_amount`。
- 代码必须兼容 Python 3.8，不使用 `match`、`|` 类型联合等 3.10+ 语法。
- 质量因子缺失时必须记录 `QUALITY_FALLBACK`，不得把降级版称为完整复刻。
