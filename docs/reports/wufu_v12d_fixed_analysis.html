结论摘要

达成项

分批成交诊断

积极信号

V12-D 使用 `WUFU_SPLIT_START` 与 `WUFU_SPLIT_DONE` 代替逐分钟日志，显著降低日志量，并保留了最终执行缺口 `gap_pct`。

待优化信号

同花顺分批缺口 P95 为 33.27%，说明少量日期存在明显成交不足或平台容量限制。后续应把固定 5 分钟分批升级为动态容量追踪。

未来验证路径

止损差异专项表

输出价格、成本、可卖数量、成交状态，逐日定位差异。

动态分批引擎

按每分钟剩余容量动态切片，直到目标缺口低于阈值。

本地分钟线复核
